The Challenge

Legacy Reporting Is Broken

A large health system's 20+ hospitals and 100+ outpatient clinics were struggling with their adverse event reporting system. Frontline clinicians faced a complex mobile-unfriendly web form, with 47 categories and 22 required fields, taking an average of more than 9 minutes to complete. In many cases, clinicians would abandon the form due to cognitive overload and frustration.

Discovery & Research

Shadowing Study

Observed 18 frontline staff filing legacy web forms to understand pain points and workflow interruptions.

Average completion time: 22 minutes

Journey Mapping

Identified emotional dip at category selection and cognitive overload leading to abandonment.

67% abandon at dropdown selection

Analytics Review

Analyzed reporting patterns and identified delays in event submission timing.

Only 31% submitted within 24h

Design Process

1

Conversation Mapping

Collaborated with patient safety SMEs to identify NLP slots and create intent taxonomy for natural language processing.

  • Conversation flowchart with branching logic
  • Intent classification system
  • Required data field mapping
2

Prototype Sprint

Built high-fidelity Figma prototype with Voiceflow integration to test voice vs text input preferences.

  • Interactive prototype with voice capabilities
  • User testing with 15 frontline staff
  • Input preference analysis
3

LLM Fine-tuning

Trained model on 8K synthetic and 2K de-identified real reports using RLHF for completion and brevity.

  • Optimized prompt library
  • Response quality metrics
  • Completion rate optimization
4

Validation Testing

Conducted Wizard-of-Oz bedside pilots on 2 medical-surgical units with 42 real reports.

  • Pilot performance dashboard
  • User feedback analysis
  • System improvement recommendations

The Solution

Conversational, Context-Aware Reporting

FLAER uses a mobile-first, voice-enabled chat interface to guide clinicians through event reporting in seconds, not minutes. The AI asks only relevant follow-ups, validates data, and routes reports in real time.
